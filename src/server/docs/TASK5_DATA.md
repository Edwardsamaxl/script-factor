## 使用方式

### 获取内置人设

```javascript
// 加载内置数据
import builtInPersonas from './data/built-in-personas.json';

const personas = builtInPersonas.filter(p => p.isPublic);
```

### 获取内置场景

```javascript
import builtInScenes from './data/scenes.json';

const scenes = builtInScenes.filter(s => s.isBuiltIn);
```
