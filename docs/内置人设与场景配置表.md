## 二、内置场景（3个）

### 场景1：过年催婚

| 字段 | 内容 |
|------|------|
| **名称** | 过年催婚 |
| **场景描述** | 家庭聚会饭桌上，七大姑八大姨围着你问东问西，从工作到感情，从工资到房子，轮番轰炸 |
| **prompt模板** | 过年家庭聚会上，长辈围着年轻人催婚，场面一度尴尬。两个人你来我往，催婚的步步紧逼，年轻人花样百出地敷衍或反击。 |
| **预期走向** | 前半段催得紧，中间年轻人反击，最后可能有妥协或话题转移 |
| **话题关键词** | 结婚、对象、年龄、抱孙子、隔壁老王家孩子 |

---

### 场景2：亲戚盘问

| 字段 | 内容 |
|------|------|
| **名称** | 亲戚盘问 |
| **场景描述** | 过年走亲戚，被七大姑八大姨围攻，从工资到买房，从对象到生娃，灵魂拷问轮番轰炸 |
| **prompt模板** | 亲戚聚会的客厅里，几个长辈围着年轻人进行年终大盘问。工资多少？买房了吗？对象呢？什么时候结婚？年轻人疲于应付。 |
| **预期走向** | 越问越深入，年轻人的敷衍越来越不走心，长辈不依不饶 |
| **话题关键词** | 工资、房子、车子、对象、存款、生娃 |

---

### 场景3：工作交接

| 字段 | 内容 |
|------|------|
| **名称** | 工作交接 |
| **场景描述** | 老员工要离职，把工作交接给新来的实习生。老员工能推就推，实习生一脸懵但很努力 |
| **prompt模板** | 办公室里，老员工正在跟实习生做工作交接。老员工说得云里雾里，能不说的就不说，实习生认真记笔记但越来越困惑。 |
| **预期走向** | 实习生问一个老员工躲一个，最后丢一堆文档让实习生自己看 |
| **话题关键词** | 需求、文档、代码、流程、找谁问、之前谁负责 |
